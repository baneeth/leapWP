================================================================================
                           TROUBLESHOOTING
================================================================================

If deployment fails:
1. Check Vercel logs (Dashboard ‚Üí Deployments ‚Üí [latest] ‚Üí Logs)
2. Verify requirements.txt exists
3. Check environment variables are set
4. Ensure DATABASE_URL is correct from Neon

If database doesn't connect:
1. Verify /admin/init-db returns success
2. Check DATABASE_URL in Vercel settings
3. Copy connection string directly from Neon panel

If pages show 404:
1. Wait 2-3 minutes, Vercel might still be building
2. Hard refresh: Ctrl+Shift+R
3. Check Vercel build logs

See VERCEL_DEPLOYMENT.md for comprehensive troubleshooting.
